This practise demonstrates how to perform basic database operations in Spring Boot using Spring JDBC with an H2 in-memory database. It covers the setup of JdbcTemplate and DataSource, as well as using SQL scripts for schema creation and data insertion.

Features

Spring Boot Integration: Simplifies application setup and configuration.

Spring JDBC: Performs CRUD operations using JdbcTemplate.

Used H2 In-Memory Database

Database Initialization:

schema.sql – Defines the database tables.

data.sql – Preloads sample data.


Technologies Used

Java 17 (or your Java version)

Spring Boot

Spring JDBC

H2 Database

Maven
